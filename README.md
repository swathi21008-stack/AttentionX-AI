# AttentionX AI - Video Repurposing Tool

## What it does
AI tool that analyzes video transcripts and automatically finds the TOP 3 most engaging moments with captions.

## Tech Stack
- Groq AI (LLaMA 3.3)
- Python
- Google Colab

## How to Run
1. Open the .ipynb file in Google Colab
2. Add your Groq API key
3. Run all cells
